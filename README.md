#  FABO 

Sitio web oficial de **FABO**, la comunidad de fútbol amateur que conecta jugadores, organiza partidos presenciales y promueve la pasión por el deporte en Buenos Aires.

---

##  Descripción

FABO es un proyecto creado para unir a futbolistas amateurs de todas las edades.  
En el sitio web podés conocer más sobre nuestra comunidad, ver fotos de los equipos, unirte a las zonas activas, consultar la indumentaria oficial y contactarnos directamente por WhatsApp o redes sociales.

---

##  Secciones principales

- **Inicio / Hero:** Presentación del proyecto FABO.  
- **Quiénes somos:** Nuestra historia y visión.  
- **Cómo funciona + Zonas:** Explicación de la dinámica de partidos y zonas de juego.  
- **Galería + Comunidad:** Imágenes y momentos destacados de los jugadores.  
- **Indumentaria:** Tienda con la ropa oficial de FABO (camisetas, shorts y conjuntos).  
- **Unite ahora:** Formulario o botón de contacto para sumarte a la comunidad.

---

##  Tecnologías utilizadas

- **HTML5**, **CSS3**, **JavaScript (Vanilla)**  
- **Tailwind CSS** para estilos y diseño responsive  
- **Firebase** (para almacenamiento, autenticación y base de datos)  
- **WhatsApp API** para contacto directo  
- **Hosting:** (por ejemplo, GitHub Pages / Vercel / Netlify)
